# USB-C-Eurorack-PSU
STUSB4500 + TPS5430 based USB-C Eurorack PSU for the JLCPCB contest.

Needs to be programmed once via I2C to request 20V and 15V as fallback.
Can deliver around 2A of current on +/-12V and 5V.
